## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Hexo clean

```shell
$ hexo clean
```

### Run server

``` bash
$ hexo server
```

### Generate static files

``` bash
$ hexo generate
```

### Deploy to remote sites

``` bash
$ hexo deploy
```
